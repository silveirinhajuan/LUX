Follow these steps to run the project locally:  

1. **Clone the repository**:  
   ```bash
   git clone https://github.com/silveirinhajuan/LUX.git
   cd LUX
   ```  

2. **Set up a virtual environment**:  
   ```bash
   python -m venv venv
   ```  


   ```bash
   python src/main.py
   ```  
